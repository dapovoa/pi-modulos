List all models available to the user.

application/json

string\|null

##### model

string\|null

200

Successful Response

##### data

array< [BaseModelCard](https://docs.mistral.ai/api/endpoint/models#operation-list_models_v1_models_get_responses_200_application-json_data_basemodelcard) \| [FTModelCard](https://docs.mistral.ai/api/endpoint/models#operation-list_models_v1_models_get_responses_200_application-json_data_ftmodelcard) >
