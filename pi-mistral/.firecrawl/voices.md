application/json

integer

Maximum number of voices to return

##### offset

integer

Offset for pagination

200

\*integer

##### page\_size

\*integer

##### total

\*integer

##### total\_pages

\*integer

application/json

Request model for creating a new voice with base64 audio.

##### age

integer\|null

##### color

string\|null

##### gender

string\|null

##### languages

array<string>

##### name

\*string

##### retention\_notice

integer

_Default Value:_`30`

##### sample\_audio

\*string

Base64-encoded audio file

##### sample\_filename

string\|null

Original filename for extension detection

##### slug

string\|null

##### tags

array<string>\|null

Successful Response

##### age

##### color

string\|null
